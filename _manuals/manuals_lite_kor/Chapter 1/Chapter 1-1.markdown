# 1.1. QRT-LITE 

QRT-LITE(Laser Irradiation Tools for Evaluation)는 Pulsed Laser를 통해 우주 방사선이 반도체 소자에 미치는 영향을 모사하여 기존 방사선 시험을 대체하여 평가를 수행할 수 있도록 도와주는 프로그램입니다.
