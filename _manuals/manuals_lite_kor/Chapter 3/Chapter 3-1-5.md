# 3.1.5. 창 크기

-	Full Screen: 전체 화면
-	Minimize: 창 모드
-	Shift: 듀얼 모니터일 경우 옆 화면으로 이동
-	Close: Save 후 종료 (Close가 아닌 비정상적인 종료 시 Save 되지 않을 수 있음)
