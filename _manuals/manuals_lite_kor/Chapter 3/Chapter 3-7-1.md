# 3.7.1. 오실로스코프(MSO)를 이용한 트리거 측정

Keysight의 오실로스코프를 사용해서 SET를 관찰하려 할 때 사용합니다. (다른 오실로스코프도 사용 가능할 것이나 확인 필요) 사용자가 설정해둔 트리거 컨디션에 따라서 트리거가 발생할 경우 설정된 경로에 파형이 기록된 파일을 발생한 시간을 파일명으로 남깁니다. 대략적인 측정 속도는 10Hz 수준입니다.
