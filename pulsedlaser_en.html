Laser Services



Error Rate Analysis

SEE evaluation quantitatively reveals device sensitivity and radiation tolerance. We observe Single Event Upset (SEU) and Single Event Latch-up (SEL) under laser irradiation and derive the cross-section (σ). Using this, we estimate LETL (laser-equivalent LET) and predict the error rate for your actual space or radiation environment.

SEREEL2

SEREEL2 is a precision, laser-based SEE test solution that overcomes the limitations of accelerator testing in space/aviation semiconductor qualification. QRT supports 630-1700 nm for SPA and TPA operation, and features auto-scan, autofocus, and chip-image/stage synchronization to quickly map even fine sensitive regions. Industrial-grade laser stability and multi-axis, high-precision motion enable flexible use from R&D to high-volume screening.

QRT-LITE (Laser Irradiation Tools for Evaluation)

QRT-LITE is QRT's in-house, laser-based SEE (Single Event Effect) software platform. In a femtosecond laser test environment, it lets you control the entire flow-setup → run → acquisition → analysis-on a single screen. You can save and recall laser conditions (wavelength, energy, etc.) for reproducibility and consistency, and synchronously capture electrical responses by integrating oscilloscopes and source meters. By mapping chip images to coordinates, you can position the stage precisely at the desired spot. With scan scheduling, fully unattended automation is also possible. Together, these capabilities deliver fast, precise reliability verification for space/aviation semiconductors.

The QRT-LITE Manual walks through everything-installation, instrument integration, mapping/scan setup, automation features, and system setup-step by step.
