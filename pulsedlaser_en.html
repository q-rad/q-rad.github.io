Laser Services



Error Rate Analysis

SEE (Single Event Effect) evaluation is a key test to quantitatively determine a device's error sensitivity and radiation hardness. By observing Single Event Upset (SEU) and Single Event Latch-up (SEL) phenomena induced by laser irradiation, we derive the Cross-Section values. These Cross-Sections are used to calculate LETL (Linear Energy Transfer Threshold), which enables us to predict the error rate in actual space or radiation environments.

Sensitive Area Analysis

QRT Laser Evaluation System



SEREEL2

SEREEL2 is a high-precision laser-based SEE (Single Event Effects) testing solution that overcomes the limitations of accelerator testing in aerospace semiconductor reliability evaluation. Supporting a wide wavelength range, it enables laser irradiation in both SPA and TPA modes, and with features like automatic scanning, autofocus, and chip image-stage synchronization, it allows easy and rapid analysis of even the smallest sensitive areas. The combination of industrial laser stability and high-precision multi-axis stage movement enables flexible responses from R&D to mass screening.
